\# Программно-аналитический комплекс для прогнозирования остатков нераспроданного жилья



Данный проект представляет собой систему автоматизированного прогнозирования временных рядов на рынке недвижимости. Реализован ансамблевый подход, объединяющий классические статистические методы и алгоритмы машинного обучения.



\## 🌟 Основные особенности

\- \*\*Интегрированные модели:\*\* Meta Prophet, ARIMA (с декомпозицией STL), Exponential Smoothing (ETS).

\- \*\*Интеллектуальный подбор параметров:\*\* Оптимизация гиперпараметров ETS с использованием байесовских алгоритмов библиотеки `Optuna`.

\- \*\*Взвешенный ансамбль:\*\* Автоматическое объединение прогнозов на основе точности каждой модели (инверсия MAE) для минимизации ошибки.

\- \*\*Модульная архитектура:\*\* Расчетная логика вынесена в отдельный модуль `forecaster.py` для удобства интеграции и защиты кода.
