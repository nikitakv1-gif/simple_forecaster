# Прогнозирование остатков жилья

Небольшой проект для прогнозирования временных рядов на рынке недвижимости.

Используются несколько моделей и их объединение для получения более точного прогноза.

---

## 📌 Что внутри

- Prophet  
- ARIMA (через STL-разложение)  
- Exponential Smoothing (ETS)  
- Optuna для подбора параметров  
- Ансамбль моделей (объединение прогнозов)

---

## ⚙️ Установка

Рекомендуется Python 3.10 или 3.11

### 1. Создать виртуальное окружение

```bash
python -m venv venv
```

Активация:

Windows:
```bash
venv\Scripts\activate
```

Mac / Linux:
```bash
source venv/bin/activate
```

---

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

> Если есть проблемы с Prophet на Windows — установи C++ Build Tools

---

## 🚀 Как использовать

1. Открой Jupyter Notebook  
2. Убедись, что `forecaster.py` лежит рядом  
3. Импортируй и запускай модели  

Для автообновления кода:

```python
%load_ext autoreload
%autoreload 2
```

---

## 📁 Структура

- `forecaster.py` — вся логика моделей  
- `Analysis.ipynb` — эксперименты и графики  
- `requirements.txt` — зависимости
